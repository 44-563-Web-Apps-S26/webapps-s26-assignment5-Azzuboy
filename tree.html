Which row breaks first when reducing the browser width?

The tree names row (the second row) breaks first. Reason: the tree names use col-md, so those columns collapse into stacked blocks below the medium breakpoint (below 768px). The letters use col-sm-2, which remain as columns until the small breakpoint (below 576px). Therefore the col-md row stacks earlier as the viewport becomes narrower.
